# Frontend Architecture

## Purpose

Define frontend technologies and UX implementation principles.

---

# Technology Stack

## Framework

Next.js

Reasons:

- modern React architecture
- excellent ecosystem
- strong AI tooling support
- scalable

---

## Language

TypeScript

Required.

No new JavaScript-only modules.

---

## Styling

Tailwind CSS

Reasons:

- maintainability
- consistency
- speed
- component friendliness

---

## Component Library

shadcn/ui

Reasons:

- accessibility
- customization
- modern design patterns

---

# Design Philosophy

The interface should feel:

- calm
- spacious
- premium
- reflective
- emotionally safe

---

# Theme System

Default Theme:

Moranta Classic

Colors:

- Deep Midnight Blue
- Soft Indigo
- Sage Green
- Moonlight Cream
- Warm Sand

---

# Additional Themes

- Forest
- Ocean
- Sunrise
- Night Sky
- Accessibility Mode

Themes should be user configurable.

---

# Accessibility

Required:

- keyboard navigation
- screen reader support
- WCAG compliance
- adjustable font sizing
- contrast validation

---

# Mobile Support

Mobile-first responsive design.

Future native applications should reuse the same design language.
