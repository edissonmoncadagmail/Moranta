# Moranta User Personas

# 1. The Reflective Seeker

## Description

Individuals interested in mindfulness, journaling, meditation, and
personal growth.

## Goals

- emotional clarity
- mindfulness
- self-awareness
- intentional living

## Needs

- calm experiences
- reflective guidance
- meaningful insights
- non-judgmental tools

------------------------------------------------------------------------

# 2. The Dream Explorer

## Description

Users deeply interested in dreams, symbolism, lucid dreaming, and
subconscious exploration.

## Goals

- track dreams
- identify patterns
- improve dream recall
- explore lucid dreaming

## Needs

- intuitive dream journaling
- symbolic exploration
- reflective analysis
- recurring pattern tracking

------------------------------------------------------------------------

# 3. The Conscious Creator

## Description

Users interested in affirmations, meditation audio, subconscious
conditioning, and intentional routines.

## Goals

- create personalized rituals
- improve mindset
- cultivate positive habits
- build intentional routines

## Needs

- affirmation tools
- audio experiences
- ambient sound systems
- creative personalization

------------------------------------------------------------------------

# 4. The Mindful Beginner

## Description

Users new to spiritual growth and mindfulness who want approachable and
grounded experiences.

## Goals

- reduce stress
- build mindfulness habits
- improve emotional awareness

## Needs

- simple onboarding
- gentle guidance
- emotionally safe experiences
- non-dogmatic language

------------------------------------------------------------------------

# 5. The Spiritual Practitioner

## Description

Experienced spiritual explorers interested in deeper reflective tools
and conscious practices.

## Goals

- deepen awareness
- expand reflection practices
- integrate mindfulness into daily life

## Needs

- advanced journaling
- intentional reflection systems
- creator content
- meaningful spiritual exploration

------------------------------------------------------------------------

# Shared User Values

Most Moranta users value:

- authenticity
- mindfulness
- emotional growth
- intentional living
- spiritual openness
- creativity
- self-awareness
- privacy
- conscious technology
