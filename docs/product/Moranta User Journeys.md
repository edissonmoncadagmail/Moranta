# Moranta User Journeys

# Journey 1 --- Dream Reflection

## Scenario

A user wakes up from a vivid dream and wants to capture it before
forgetting details.

## Flow

1.  User opens Dream Journal
2.  Records voice or text entry
3.  Adds emotional tags
4.  AI asks reflective questions
5.  User reflects independently
6.  AI provides possible symbolic observations
7.  Timeline stores recurring themes over time

## Emotional Goal

The experience should feel:

- calm
- reflective
- non-judgmental
- personally meaningful

# Journey 3 --- Affirmation Creation

## Scenario

A user wants to create a personalized affirmation experience for
confidence and emotional grounding.

## Flow

1.  User selects intention
2.  AI helps draft affirmations
3.  User edits and personalizes language
4.  User records voice or generates narration
5.  User layers ambient audio
6.  Session saved into playlist
7.  User listens daily or before sleep

## Emotional Goal

The experience should feel:

- empowering
- calming
- intentional
- creative
- deeply personal

------------------------------------------------------------------------
